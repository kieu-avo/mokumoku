## 事業をエンジニアリングしよう提案編の回答は以下に記述してください
```
■選択した事業側の課題
サービス登録者数の内、男性60%に対して、女性は40%。一方で、サービス内のもくもく会に参加した人の比率は、男性90%：女性10%と大きな差が出ています。もっと女性が使いやすいようなサービス設計にする必要があるのではないか？

■提案内容
女性限定のもくもく会を開催する
- 仕事、育児、家事と忙しく参加したくても参加ができない女性が多いと思う。参加しやすい時間や曜日を聞いてから、イベントを開催する。
    例：5人以上集まれそうな場合、イベントを開催。最初は集まりやすい曜日と時間を固定にして毎週開催する。 
- 女性に限定することで女性ならではの悩み（育児・キャリア等）などを発散できる場にもなる。

■実装方針
- プロフィールに性別の選択ができるフィールドを追加する 
  →「女性」を選択すると、参加しやすい曜日と時間の記入ができる 
- もくもく会作成に、「一般」と「女性限定」の選択を追加する
- 「女性限定」の検索ができるようにする

```
